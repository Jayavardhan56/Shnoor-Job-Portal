🚀 SHNOOR Job Recruitment Portal

An advanced AI-powered recruitment and hiring management platform built to simplify the complete recruitment lifecycle — from job posting and candidate applications to ATS-based resume evaluation and recruitment workflow management.

The platform provides intelligent hiring tools, role-based dashboards, analytics, and a modern responsive interface for recruiters, administrators, and job seekers.
